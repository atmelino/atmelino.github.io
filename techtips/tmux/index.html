tmux

tmux is a terminal multiplexer that allows starting a process on a remote server and logging out from the server without ending the process.
This can be useful, for example, to start a machine learning process on a remote server and keeping it running.

Project Web Sites

https://github.com/tmux/tmux/wiki

Installation

sudo apt install tmux

Usage

Detach:
putting the session into the background without ending it.
CTRL-B-D

Attach:
After logging on to the server, bringing the tmux session into the foreground again.

tmux attach-session -t 0

Scroll

Enter scroll mode:
CTRL-B-D

End scroll mode:
Q

Enable Mouse Input

Press CTRL-B and the : to enter command mode

Type
setw -g mouse on
This allows switching between panes by clicking on the pane number in the statusbar

Set mouse on automatically:

sudo nano ~/.tmux.conf
add the line
setw -g mouse on
https://www.fosslinux.com/80828/how-to-use-tmux-mouse-mode.htm

Autostart multiple tmux

create batch file, content:
tmux new-session 'top'\; new-window 'btop'\; new-window -c /media/data/github/learn/ \; new-window -c /media/data/github/learn/ai/practice/keras/image/ \;
https://stackoverflow.com/questions/8537149/how-to-start-tmux-with-several-windows-in-different-directories?rq=3
