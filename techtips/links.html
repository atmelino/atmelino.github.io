CONNECTING RASPBERRY PI WITH LAPTOP/HOST MACHINE

Method: assign static IP addresses to both Raspberry Pi as well as laptop.

On laptop:
use GUI to set
address 192.168.0.50
netmask 255.255.255.0

On Raspberry Pi:
nano /etc/dhcpd.conf
interface eth0
static ip_address=192.168.1.51
#static ip6_address=fd51:42f8:caae:d92e::ff/64
static routers=192.168.1.1
static domain_name_servers=192.168.0.1 8.8.8.8 fd51:42f8:caae:d92e::1
metric 300

interface wlan0
metric 200

check with
ifconfig
connect directly with Ethernet cable
SSH access:
ssh pi@192.168.0.51
